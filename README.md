# rapflag
Bitfinex Exporter for your Taxman.

### set your key in
```
 ./etc/config.yml
```
sample
```
---
api_key: ''
secret: ''
```
### run as follows
```
ruby rapflag.rb
ruby rapflag.rb -c (clean option to show daily income and daily value of your wallet)
```
Enjoy your reports. More to come. Feedback always welcome.
